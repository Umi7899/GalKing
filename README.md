# GalKing - AI 日语学习助手

GalKing 是一款致力于通过 AI 技术提升日语学习体验的移动应用。它不仅提供传统的语法和词汇训练，还集成了一位富有活力的 AI 教练，为您提供个性化的点评和指导。

## 🌟 核心功能

- **AI 教练 (Sakura)**：一位元气满满的日本女高中生 (JK) 老师，根据您的水平提供自适应難度的日语点评。
- **自适应难度**：AI 会根据您的 Lesson 进度和 Level 自动调整交流的日语复杂度（N5-N3+）。
- **短期记忆与出勤分析**：教练会记住您上次的表现，并根据您的打卡记录（全勤/偶尔断更/久违回归）调整语气。
- **智能解锁逻辑**：结合 AI 评估和实际正确率（连续3次 >85%）自动解锁新课程。
- **学习历史与导出**：持久化保存所有 AI 点评，支持回顾和一键分享。
- **全语音/多维度训练**：包含语法、词汇、句子解析等多种练习模式。

## 🛠️ 技术栈

- **Frontend**: React Native, Expo
- **Database**: SQLite (Expo SQLite)
- **AI Integration**: OpenAI 兼容协议 (支持 DeepSeek, GPT 等)
- **Language**: TypeScript
